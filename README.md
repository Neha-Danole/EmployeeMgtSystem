 ## Employee Management System (Full Stack) :

This is a Full Stack Employee Management System built using Spring Boot (Backend) and React (Frontend). The application provides secure role-based authentication and complete employee data management with a modern and scalable architecture.


 ## 🧱 Tech Stack:

 ## 1.Backend:
 - Java
 - Spring Boot
 - Spring Security
 - Spring Data JPA
 - JWT
 - MySQL

## 2.Frontend:
 - React.js
 - Axios
